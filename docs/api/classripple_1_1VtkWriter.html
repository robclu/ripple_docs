class ripple::VtkWriter : public ripple::MultidimWriter¶

The VtkWriter type implements the multidim writer interface to write data to a file in the Vtk format.

Public Types

using DimSizes = MultidimWriter::DimSizes¶

Defines the type of the dimension sizes.

Constructor which sets the base name of the file to write, as well as the name of the data, and the names of any elements to write.

Parameters

  • base_filename: The base name of the file to write to.

  • data_name: The name of the data.

~VtkWriter()¶

Destructor which closes the file.

VtkWriter(const VtkWriter &other) noexcept¶

Copy constructor to create another writer from this writer.

This copies the parameters from the other writer, but does not open or write any data for the writer.

Parameters

  • other: The other writer to copy from.
