template<typename T, typename Rows, typename Cols, typename Layout>
struct ripple::MatImpl¶

The MatImpl class implements a matrix class with polymorphic data layout.

The data for the elements is allocated according to the layout, and can be contiguous, owned, or strided.

Note

This class should not be used directly, use the Mat aliases.

Template Parameters

  • T: The type of the elements in the matrix

  • Rows: The number of rows for the matrix.

  • Cols: The number of colums for the matrix.

  • Layout: The type of the storage layout for the matrix.
