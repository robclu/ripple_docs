class ripple::MultidimWriter¶

The MultidimWriter class defines an interface for writing data to a file for multi dimensional data types.

Subclassed by ripple::VtkWriter

template<typename T, size_t Dims, typename ...Args>
auto write(const Tensor<T, Dims> &data, const ElementNames &element_names, Args&&... args) -> void¶

Writes the elements with the given names from the data tensor.

Note

This will cause a compile time error if the data dos not implement the Printable interface, or is an arithmetic type.

Parameters

  • data: The data to write.

  • element_names: The names of the elements to write.

  • args: Additional arguments for getting an element.

Template Parameters

  • T: The type of the tensor data.

  • Dims: The number of dimensions for the tensor.

  • Args: The type of additional arguments.
