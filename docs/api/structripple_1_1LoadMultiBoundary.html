struct ripple::LoadMultiBoundary¶

Functor which applies a loader to call the boundary loading function.

This can be passed to a graph to be applied to a tensor.

Public Functions

template<typename IteratorA, typename IteratorB, typename Loader, typename ...Args>
auto operator()(IteratorA &&it_a, IteratorB &&it_b, Loader &&loader, Args&&... args) const noexcept -> void¶

Loads the global boundary data for the iterator using theloader.

Note

The loader must implement the BoundaryLoader interface.

Parameters

  • it: The iterator to load the boundaries for.

  • loader: The loader which defines how to load the boundaries.

  • args: Additional arguments for the loading.

Template Parameters

  • Iterator: The type of the iterator.

  • Loader: The type of the loader.

  • Args: The types of the arguments.
